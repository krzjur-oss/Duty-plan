# Duty Plan

## Project Description
Duty Plan is a project management tool designed to help teams organize, schedule, and track their tasks efficiently. It provides an intuitive interface and a robust backend to manage various aspects of project planning and execution.

## Features
- **Task Management**: Create, edit, and delete tasks.
- **Scheduling**: Set deadlines and reminders.
- **Team Collaboration**: Assign tasks to team members and communicate updates.
- **Real-time Analytics**: Monitor progress with visual dashboards.
- **Customizable Workflows**: Tailor the process to fit your team's needs.

## Installation Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/krzjur-oss/Duty-Plan.git
   cd Duty-Plan
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment Variables**:
   Create a `.env` file and set necessary configurations.
4. **Run the Application**:
   ```bash
   npm start
   ```

## Usage Guide
- To create a new task, navigate to the 'Tasks' section and click on 'Add Task'.
- To view analytics, go to the 'Dashboard' where you can see progress overviews.

## Technical Details
- **Language**: JavaScript
- **Framework**: Node.js with Express
- **Database**: MongoDB for data storage
- **Frontend**: React.js for building user interfaces
